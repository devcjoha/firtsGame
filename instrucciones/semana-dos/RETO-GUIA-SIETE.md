# 📱 ¡Tu Juego en el Bolsillo! 📱

Tu código ya es 100% compatible con celulares! El desafío de hoy es lograr ver y jugar el juego desde tu propio teléfono físico antes de que termine la clase.

Abre la terminal de tu computadora donde corre el juego y apágalo presionando `Ctrl + C`.

Enciéndelo con el comando de transmisión abierta escribiendo:

```Bash
npm run dev -- --host
```
Busca la línea que dice Network (ejemplo: `http://192.168.1.15:5173/`).

Saca tu celular, conéctalo al mismo Wi-Fi de la computadora, abre el navegador y escribe esos números.

🏆 Condición de victoria del reto: ¡Dar la vuelta al teléfono de forma horizontal, tocar la pantalla y lograr que el cuadrado azul salte en tus propias manos!
